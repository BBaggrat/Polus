# Release 1.0 scope freeze

## Входит в релиз 1.0

- Telegram Bot `/start` + WebApp-кнопка.
- Mini App.
- Живой дневник исследования.
- `HIDDEN`-режим.
- `OPEN_PVP`-режим.
- Текущая RPS/PvP-стычка.
- База.
- Ресурсы.
- Апгрейды базы.
- Снаряжение.
- Карта, фрагменты, маршруты.
- События.
- Цепочки событий.
- Монстры как text encounters.
- Находки/discoveries.
- Логирование.
- Базовая аналитика.
- Smoke/regression checklist.
- Production deploy instructions.

## Не входит в релиз 1.0

- Монетизация.
- Кланы.
- Рынок.
- Торговля между игроками.
- Полноценная визуальная карта.
- Полноценная PvE-боевка.
- Большой сюжетный режим.
- Сезоны рейтинга.
- Сложный крафт.
- Дерево навыков.
- Push-уведомления.
- Сложная админ-панель.

## Правило изменения scope

До релиза 1.0 допускаются только bugfix, copy polish, deploy fixes, smoke/regression fixes и небольшие правки баланса, если они не меняют экономику радикально.
