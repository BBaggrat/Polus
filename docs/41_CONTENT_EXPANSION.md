# Расширение контента

Stage 4 добавляет контент поверх существующего Stage 2 каталога.

## HIDDEN

Добавлено:

- 8 записей движения;
- 8 событий добычи;
- 8 объектных событий;
- 6 аномалий;
- 6 событий существ;
- 4 фрагмента карты;
- 4 воспоминания базы.

## OPEN_PVP

Добавлено:

- 6 следов PvP;
- 4 PvP-встречи;
- 3 aftermath-события;
- 3 risk/reward-события.

## RETURN / FAILED

Добавлено по 4 варианта текста для успешного возвращения и провала вылазки.

## Технически

Каталог доступен через:

```http
GET /api/events/catalog
```

Ответ содержит totals, раздельные counts по hidden/open-pvp и counts по типам encounter.
