# PvP release check

## Проверено по коду и тестам

- `PISTOLS` работает.
- `RIFLE` работает.
- `SHOTGUN` работает как текущий shotgun/MG-like многопульный профиль.
- Направления выстрела работают.
- Направления уворота работают.
- Расчет урона покрыт `DuelEngineTest`.
- Бой завершается.
- Победа/поражение дают награды.
- Результат логируется через `duel_finished` и `pvp_finished`.
- `OPEN_PVP` запускает существующий duel queue flow через `startPvpDuel`.
- `HIDDEN` не запускает принудительный PvP.

## Найденные баги

- Блокирующих багов не найдено.

## Ограничения

- Итог PvP не привязан обратно к конкретной цепочке исследования.
- `SHOTGUN` исторически может восприниматься как MG из старых требований; в коде релиза оружие называется `SHOTGUN`.
