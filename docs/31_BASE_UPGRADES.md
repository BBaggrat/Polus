# Улучшения базы

| Улучшение | Уровни | Стоимость уровней | Эффект |
|---|---:|---|---|
| Склад | 3 | `0/2/0`, `4/2/0`, `8/3/1` | минус 10/20/30% к потерям добычи |
| Картографический стол | 3 | `2/1/0`, `5/2/0`, `8/3/1` | +10/20/30% к шансу фрагмента |
| Укреплённый настил | 3 | `2/2/0`, `5/3/0`, `8/4/1` | минус 5/10/15% к потере HP |
| Сушилка припасов | 3 | `1/2/0`, `4/3/0`, `7/4/1` | шанс 20/35/50% получить +1 припас из loot |
| Оружейный верстак | 1 | `5/3/1` | открывает улучшение снаряжения |
| Оберег у входа | 3 | `2/1/1`, `5/2/1`, `8/3/2` | минус 5/10/15% к аномальным последствиям |

Стоимость записана как `лом/припасы/смола`.

Покупка атомарно проверяет запас игрока, списывает стоимость и повышает уровень. При нехватке ресурсов возвращается понятная ошибка и событие `base_upgrade_failed_not_enough_resources`.

Первый уровень склада доступен из стартовых трёх припасов, чтобы новый цикл можно было увидеть без предварительного гринда.
